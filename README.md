# ElevateLabsTask1
# Python CLI Calculator

## Description
This is a simple **Command-Line Interface (CLI) Calculator** written in Python.  
It supports the following basic operations:
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)

The program loops continuously until the user chooses to exit.
